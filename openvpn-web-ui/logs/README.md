# Logs Directory

This directory contains application logs generated by Winston logger.

## Log Files

- `app.log` - Main application log file
- `error.log` - Error-level logs only (if configured)

## Log Rotation

Logs are automatically rotated when they reach:
- **Max size:** 10MB per file
- **Max files:** 5 files kept

Older logs are automatically deleted when the limit is reached.

## Log Levels

- `error` - Error messages
- `warn` - Warning messages
- `info` - Informational messages
- `http` - HTTP request logs
- `debug` - Debug messages (if enabled)

## Viewing Logs

```bash
# Real-time log viewing
tail -f logs/app.log

# Last 200 lines
tail -n 200 logs/app.log

# Search for errors
grep "error" logs/app.log
```

## Via Web UI

Access the Logs page at http://your-server:3000/logs.html after logging in.

---

**Note:** Logs are automatically created when the application starts. This directory must be writable by the application user.
